# Dotfiles

## Supported Platform
- macOS
- Linux (tested on Raspbian Lite)

## Just run
```
curl -sSL bit.do/spywhere-dotfiles | sh
```

or

```
curl -sSL git.io/JvZB8 | sh
```

## Development

To run the setup without updating use

```
sh install.sh -l
```

To simulate a remote setup use

```
cat install.sh | sh -s -- [args...]
```
